Roteiros de Aulas Práticas

Seguem roteiros de aulas práticas sobre alguns capítulos do livro. Os roteiros incluem um passo a passo detalhado e, por isso, podem ser seguidos por alunos de forma independente, mesmo que eles não tenham experiência nas tecnologias envolvidas.

Para facilitar a sua realização, todos os roteiros estão armazenados no GitHub.

  • Cap. 7: Arquitetura
    • Microsserviços, usando Node.js, REST, gRPC e Docker
    • Publish/Subscribe, usando RabbitMQ
  • Cap. 8: Testes
    • Desenvolvimento Dirigido por Testes (TDD), usando um exemplo do livro de TDD do Kent Beck
    • Testes end-to-end, usando Cypress
  • Cap. 9: Refactoring
    • Roteiro prático de refactoring em JavaScript
  • Cap. 10: DevOps
    • Integração Contínua, usando GitHub Actions
Todos os direitos reservados. Versão apenas para uso pessoal.
